# Appendices

Use these appendices as reference material while implementing or debugging the compiler.

- `appendix-a-debugging-assembly.md` — practical debugging workflow for generated assembly and runtime failures
- `appendix-b-assembly-generation-tables.md` — compact ABI/register/alignment and instruction-pattern reference

These appendices are especially useful from chapter 9 onward, when ABI, calling convention, and backend errors become common.
