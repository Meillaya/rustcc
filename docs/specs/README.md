# Rust C Compiler SRS Package

This directory is the normative software requirements package for the book-aligned compiler project.

It defines **what the compiler must do** and **what evidence should prove it**, without dictating every implementation detail.

## Normative words

- **SHALL** = mandatory
- **SHOULD** = strong recommendation
- **MAY** = optional
