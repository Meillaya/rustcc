# Language Subset by Chapter

| Chapter | Feature growth | Main semantic pressure |
|---|---|---|
| 1 | minimal function + return literal | basic syntax acceptance |
| 2 | unary expressions | precedence and nesting |
| 3 | binary arithmetic | associativity and arithmetic lowering |
| 4 | logical/relational operators | short-circuiting and comparisons |
| 5 | locals and assignment | scope and lvalues |
| 6 | `if` and `?:` | branching semantics |
| 7 | blocks | nested scope/shadowing |
| 8 | loops | control-flow validity |
| 9 | functions/calls | ABI and signatures |
| 10 | globals/storage class | linkage and data emission |
| 11 | `long` | width-sensitive semantics |
| 12 | `unsigned` | modular arithmetic and conversions |
| 13 | floating point | SSE/XMM behavior and mixed conversions |
| 14 | pointers | indirection and address semantics |
| 15 | arrays | layout and scaled arithmetic |
| 16 | chars/strings | byte semantics and literal storage |
| 17 | `sizeof`, `void`, `void*` | completeness and size reasoning |
| 18 | structs/unions | aggregate layout and ABI |
| 19 | optimization | semantics-preserving transformation |
| 20 | register allocation | liveness, spills, coalescing |
