# Research Pack

This directory collects the external references most useful for building the compiler correctly.

## Reading strategy

Use sources in this order:

1. **primary/normative** sources for language, ABI, ISA, assembler, and toolchain behavior
2. **implementation guides** for practical workflow and pass design
3. **theory/textbook** sources for algorithms and proofs of correctness
4. **reference projects** for design comparison and sanity checks
