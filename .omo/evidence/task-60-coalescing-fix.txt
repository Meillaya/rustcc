Summary
-------
- Removed the broad dividend setup peephole from src/codegen/regalloc/rewrite.rs: the old cleanup_dividend_setup/dividend_setup_replacement functions are gone.
- Removing that behavior entirely caused the optimized chapter20 default coalescing gate to fail on briggs_dont_coalesce (8 register moves, max 7). To preserve the required gate without hiding the old broad peephole in rewrite.rs, the required destructive-division copy collapse is isolated in src/codegen/regalloc/division_copy.rs with a narrower safety guard: it refuses to move the saved dividend write across the intervening divisor-setup instruction if that instruction mentions the saved register.
- Split Task60 size regressions into focused modules:
  * src/codegen/regalloc/abi_liveness.rs owns call/return liveness metadata helpers extracted from allocate.rs.
  * src/codegen/regalloc/graph_pseudos.rs owns pseudo-node discovery extracted from graph.rs.
  * src/codegen/fixup/split.rs owns fixup split rules extracted from fixup.rs.
  * src/codegen/replace_pseudos/split.rs and src/codegen/replace_pseudos/move_split.rs own replace-pseudos split rules extracted from replace_pseudos.rs.
  * src/codegen/xmm.rs owns the shared XMM binary classifier used by fixup, replace_pseudos, regalloc rewrite, and coalescing.
- No test/harness edits: git diff -- tests is empty.
- No source path/test-name/system-C/interpreter bridge hits in src.
- R10/R11 and XMM14/XMM15 remain reserved/non-allocatable; scans still route allocation through RegisterClass::all_hardregs/contains, with R11 only in scratch reservation.
